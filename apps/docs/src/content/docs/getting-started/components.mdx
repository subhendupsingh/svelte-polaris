# Components Overview

Svelte Polaris provides a comprehensive set of components that follow Shopify's design system. All components are built for Svelte 5 and provide full TypeScript support.

## Installation

```bash
npm install svelte-polaris
```

## Basic Setup

```svelte
<script>
  import { PolarisAppProvider, Button, Card } from 'svelte-polaris';
</script>

<PolarisAppProvider>
  <Card>
    <Button variant="primary">Get started</Button>
  </Card>
</PolarisAppProvider>
```

## Actions

Components that trigger actions or navigation.

<CardGrid>
  <Card title="Button" icon="rocket">
    Primary component for triggering actions like form submission or navigation.
    
    ```svelte
    <Button variant="primary">Save changes</Button>
    ```
    
    [View Button docs →](/getting-started/button)
  </Card>
  
  <Card title="ButtonGroup" icon="puzzle">

    ```svelte
    <ButtonGroup>

    </ButtonGroup>
    ```

  <Card title="ActionList" icon="list-format">
    Displays a list of actions or options in a menu format.
    

  <Card title="ActionMenu" icon="bars">

## Tables & Lists

Components for displaying structured data.

<CardGrid>
  <Card title="IndexTable" icon="table">
    Data table with sorting, selection, and bulk actions.
    
    ```svelte

      resourceName="products"
      itemCount={products.length}
      headings={headings}
    ```

## Getting Started

1. **Install the package**: `npm install svelte-polaris`
2. **Import components**: Import only the components you need
3. **Wrap with provider**: Use `PolarisAppProvider` at your app root
4. **Start building**: Combine components to create your interface

## TypeScript Support

All components include full TypeScript definitions:

```typescript
import type { ButtonProps, CardProps } from 'svelte-polaris';

interface MyComponentProps {
  buttonProps: ButtonProps;
  cardTitle: string;
}
```
